# 投票系統

## 專案概述

本專案是一個投票系統，包含前端和後端部分。

### 後端 (BE)

- **`vote-admin`**：管理後台服務，運行在端口 `8080`
- **`vote-website`**：投票網站服務，運行在端口 `8081`

### 前端 (FE)

- **`voting-web-vue`**：前端後台網站 Vue.js 專案
- **`voting-user-vue`**：投票網站 Vue.js 專案

## 後端設置

### 1. clone project

```bash
git clone <後端倉庫 URL>
cd <後端專案目錄>
```

### voting-web-vue
cd ./fe/voting-web-vue
### install
npm install
### server
npm run dev

### voting-user-vue
cd ./fe/voting-user-vue
### install
npm install
### server
npm run dev
